This release contains the following directories:

lib/
bin/
samples/
doc/

Description of contents:

lib      - contains the compiled java classes and the compiled native
           libraries for JMF.
bin      - contains scripts to launch a few sample applications.
samples  - contains source code for the sample applications.
doc      - installation and setup instructions.

Start off by opening the doc/index.html file in a browser. This contains
links to most of the above and setup instructions.
